Description

This project is a command-line To-Do List Application developed using Python. It helps users manage and organize their daily tasks efficiently.

Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Remove all tasks
- User-friendly menu-driven interface

Technologies Used

- Python 3

How to Run

1. Download the project files.
2. Open the project in VS Code.
3. Open Terminal.
4. Run the following command:

python3 CODESOFT.py
